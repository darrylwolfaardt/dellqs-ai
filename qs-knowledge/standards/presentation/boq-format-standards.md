# BOQ Format Standards

## Document Structure
- Cover page with project details
- Table of contents
- Summary of pricing
- Detailed measured works
- Provisional sums
- Contingencies

## Formatting Rules
- All quantities to 2 decimal places
- Rates to nearest Rand
- Page numbering: Section-Page (e.g., 3-12)
- Element codes per standard classification

## Font & Layout
- Body: Arial 10pt
- Headers: Arial 12pt Bold
- Tables: Gridlines visible, alternating row shading
